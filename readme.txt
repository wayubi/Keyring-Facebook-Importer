A Facebook importer to use with the Keyring and Keyring Social Importers WordPress plugins.

== Description ==

Own the data that you create. Install the Keyring and Keyring Social Importers plugins, and then use Keyring Facebook importer to save copies of your Facebook photos, photo albums, and status updates to an easy-to-browse (and easy to share) format.

After installing, visit Import > Facebook to get started.

When you set up your Facebook App connection, your app will need to have the permissions "read_stream" and "user_photos."
